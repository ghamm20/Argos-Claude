# ARGOS Doctrine

ARGOS is a local-first AI workstation that runs fully from removable media. No install footprint. No registry writes. No cloud calls. Plug in, run, unplug — host machine is identical to before.

## What ARGOS Is
- A USB-resident, host-non-persistent local AI workstation
- A memory-aware assistant with citation-default reasoning
- A personal intelligence operating system

## What ARGOS Is Not
- A SaaS clone, AI wrapper, or roleplay app
- A cloud-dependent product with offline mode
- A fake AGI or sentience simulator

## Strategic Position
Consumer product. Path B build: 4 months to v1.5 with rigor infrastructure from week 2.
